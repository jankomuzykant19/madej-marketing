# bartlomiej-madej

Prosta aplikacja portfolio (hello world) z frontendem i backendem w Next.js.

## Lokalnie

```bash
npm install
npm run dev
```

Aplikacja: http://localhost:3000  
API: http://localhost:3000/api/hello

## Deploy na Vercel

1. Wypchnij repozytorium na GitHub.
2. Wejdź na [vercel.com](https://vercel.com) i połącz repozytorium.
3. Vercel wykryje Next.js automatycznie — kliknij **Deploy**.

Alternatywnie z CLI:

```bash
npm i -g vercel
vercel
```
